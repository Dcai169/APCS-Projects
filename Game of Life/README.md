# The Game Of Life
### Keyboard Shortcuts
- Q: Quit Program
- W: Step Forward
- E: Repaint
- R: Reset Timer Speed
- A: Toggle Grid Lines
- S: Randomize World
- D: Clear World
- F: Print toString()
***
- Space Bar: Toggle Timer
- Click: Toggle Cell Alive/Dead
- Scroll: Increment/Decrement FPS
***
- 1: Load Shape 1
- 2: Load Shape 2
- 3: Load Shape 3
